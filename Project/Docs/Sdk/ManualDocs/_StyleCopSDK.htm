StyleCop SDK
 

Welcome to the StyleCop SDK documentation. StyleCop is a code style and consistency analysis tool for C#. The tool integrates into the Visual Studio IDE, or can be run on the command line. StyleCop ships with a default set of rules enforcing common style guidelines.

This SDK documentation describes how to write custom StyleCop rules, and how to integrate StyleCop into custom build tools.
